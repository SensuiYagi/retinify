## Why retinify?
retinify is an advanced AI-powered stereo vision library designed for robotics. It enables real-time, high-precision 3D perception by leveraging GPU and NPU acceleration.
- üåê **Open Source**: Fully customizable and freely available under an open-source license.
- üî• **High Precision**: Delivers real-time, accurate 3D mapping and object recognition from stereo image input.
- üí∞ **Cost Efficiency**: Runs using just cameras, enabling depth perception with minimal hardware cost.
- üé• **Camera-Agnostic**: Accepts stereo images from any rectified camera setup, giving you the flexibility to use your own hardware.
